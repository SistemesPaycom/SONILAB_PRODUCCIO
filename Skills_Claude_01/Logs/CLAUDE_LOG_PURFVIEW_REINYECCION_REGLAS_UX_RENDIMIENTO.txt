3. OBJECTIU A — PURFVIEW .EXE INTEGRATION
──────────────────────────────────────────
CANVIS a transcription.processor.ts:

  3.1 Afegit `--output_dir outDir` als arguments del .exe:
      - Abans: el .exe escrivia al seu propi directori o al del vídeo
      - Ara:   forcem output al directori controlat (outDir = MEDIA_ROOT/projects/<id>/whisperx)
      - L'argument mediaPath segueix sent l'últim (requerit pel .exe)

  3.2 Reordenat srtCandidates per prioritzar outDir:
      - Prioritat 1: outDir (on hem dit --output_dir)
      - Prioritat 2: mediaDir (directori del vídeo, fallback)
      - Prioritat 3: exeDir (directori del .exe, fallback legacy)

  3.3 Exit codes millorats:
      - 3221226505 (0xC0000409): STATUS_STACK_BUFFER_OVERRUN — CRT cleanup crash
      - -1073740791: mateixa cosa en signed int
      - Qualsevol exit code < -1: considerat crash-on-exit (no error real)
      - exit code 1: error real (model no trobat, CUDA fail, etc.)
      - En tots els casos: busquem SRT abans de decidir si és error
