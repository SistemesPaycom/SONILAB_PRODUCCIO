4. OBJECTIU B — REINJECCIÓ EN REGLES INTERNES
================================================================================

Flux complet:
  1. .exe genera SRT → es guarda com baseName.raw.srt (backup)
  2. S'executa: python srt_postprocess.py --input raw.srt --output final.srt --subtitle-edit-compat
  3. srt_postprocess.py:
     - Parseja SRT → llista de dicts {start, end, text}
     - Aplica apply_postprocessing() del postprocessor.py existent
     - fix_text_artifacts (apòstrofes, números, clítics)
     - balance_lines (equilibrar 2 línies)
     - split_long_lines (dividir cues llargs)
     - fix_casing (capitalitzar)
     - add_periods (punt si pausa llarga)
     - --subtitle-edit-compat: NO merge_short_lines (manté cues separats)
  4. Si postprocessador falla → fallback al SRT raw (no bloqueja)
  5. SRT final es guarda a MongoDB

6. OBJECTIU D — RENDIMENT REAL (INP / CLS)
================================================================================

DIAGNÒSTIC (DevTools):
  - LCP ≈ 0.86s → OK
  - CLS ≈ 0.37 → massa layout shift
  - INP ≈ 6440ms → interactivitat molt dolenta
  - Hot paths: pointer events, canvas.w-full.h-full.block, button.absolute

CAUSA ARREL IDENTIFICADA:
  currentTime com a state (useState) → setCurrentTime cridat a cada onTimeUpdate
  del vídeo (30-60fps) → tot el component VideoSubtitlesEditorViewInner rerendera
  → cascade: linkedSegmentsWithDiff recalculat → SubtitlesEditor rerendera →
  tots els SegmentItem rerenderen → 300+ DOM nodes actualitzats 60x/s

SOLUCIÓ IMPLEMENTADA:

  6.1 currentTime throttle (VideoSubtitlesEditorView.tsx):
      - currentTimeRef.current = valor real (actualitzat a cada frame)
      - setCurrentTime (state) NOMÉS s'actualitza:
        * Quan canvia el segment actiu (per marcar-lo)
        * Cada 250ms per al toolbar/timecodes
      - Resultat: ~4 rerenders/s en lloc de 60

  6.2 React.memo (SegmentItem, SubtitlesEditor, WaveformTimeline):
      - SegmentItem no rerendera si les props no canvien
      - Amb currentTime throttled, la majoria de SegmentItems NO reben props noves
      - Només l'actiu (isActive canvia) es rerenderitza

  6.3 Stable callbacks (VideoSubtitlesEditorView + SubtitlesEditor):
      - handleSegmentFocus: useCallback amb dep [isEditing]
      - handleInsertBefore/After: useCallback amb dep [onInsert]
      - handleNavigate: useCallback amb dep [segments, onSegmentClick]
      - Inline `(id) => fn(id)` eliminats del JSX
      - playerProps.onTimeUpdate: handleTimeUpdateThrottled (estable)

  6.4 RAF throttle (usePanelResize.ts):
      - mousemove durant resize limitat a 1 update per frame
      - cancelAnimationFrame al mouseUp

  IMPACTE ESPERAT:
    - INP: de 6440ms a <500ms (rerenders 15x menys freqüents)
    - CLS: millora per menys layout recalculations durant reproducció
    - Drag de subtítols: ja tenia RAF, ara sense rerenders innecessaris
    - Resize de panells: RAF throttle elimina jank
